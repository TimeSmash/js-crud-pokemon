Welcome to Pokemon Lists!

This exercise displays all CRUD actions of JS using info pulled from PokeAPI. The actions are:

  • Create: Ability to add brand new Pokemon based on user input
  • Read: Can see list of all pre-established Pokemon at beginning, then the subsequent edits to that list. Can also view stats for all Pokemon in JSON.
  • Update: Alter some information about a Pokemon via a form
  • Delete: Ability to delete Pokemon from list
